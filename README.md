# TaskManager

Aplicação web para gerenciamento de tarefas, permitindo criar, visualizar, atualizar e remover tarefas de forma simples e eficiente.

Tecnologias utilizadas
• Go (Golang)
• Angular
• PostgreSQL

Funcionalidades
• Criar tarefas
• Listar tarefas
• Atualizar tarefas
• Deletar tarefas

Melhorias futuras
• Autenticação com JWT
• Filtros de tarefas
• Deploy da aplicação
• Melhorias na interface
